# Contributing

**This project does not accept pull requests.**

It is developed as a personal project and published under the MIT license for others to
use, fork, and adapt. PRs will be closed unmerged.

You are welcome to:

- **Fork** the repository and build on it (that's what the MIT license is for).
- **Open an issue** to report a bug or ask a question — no promise of a response, but
  reports are appreciated.

Thanks for understanding.
